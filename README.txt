Context menutrail
=================

Set the active menu trail as a context reaction.

This module contains a plugin duplicated from the contrib context plugin
context_reaction_menu and then patched with the work at:
https://drupal.org/comment/7948081#comment-7948081

- Supports translated menu items.
- Provides a variation of breadcrumbs as well.

Installation
------------

Install and enable as usual

Configure
---------

1. Visit admin/structure/context/add
2. Create a condition, ie. a specific content type
3. Add a Menu trail reaction and select the menu item you want active
